# Car_Price_Trends_Mohammed
The data set used contains 50,000 observations and 7 features of used cars, spanning from 1984 to 2022. The features included are: Manufacturer, Model, Engine Size, Fuel Type, Year of Manufacture, Mileage, and Price. Each feature has a value for every row, none of which were NA values. 

Data set source: https://www.kaggle.com/datasets/minahilfatima12328/car-sales-info
